# KickSquad

# 🚴🏻‍YOLOv5를 활용한 실시간 전동킥보드 운전자 모니터링 및 주차 인식 서비스


*'KickSquad🚴' is a Electric kickboard law violation detection system.*

 *It is a system that finds two or more people on the kickboard or who do not wear a helmet.*

 
 ## ⚡TEAM⚡
